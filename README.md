# PlainsAndDepressionsAPI
Проведение подсчетов впадин по запросам

Дан двумерный массив (матрица), содержащий только 0 и 1. 
Матрица представляет землю, где 0 — впадина, 1 — равнина. 
Впадины могут быть разного размера, они организуются путем соединения соседних ячеек сверху или снизу, а не по диагонали. 
Необходимо написать функцию, которая будет возвращать размеры всех долин в порядке возрастания.
